About the Game

WriteNor is an immersive simulation horror game. You take on the role of a developer tasked with debugging a corrupted system. However, the boundaries between the software and reality begin to blur.

The game features a fully diegetic interface. There are no pause menus or HUDs. Every interaction happens physically within the game world through monitors and terminals.

"DELETE. REWRITE. SURVIVE."

Key Features

Console Commands

Use the terminal to manipulate the environment. Delete locked doors, change game variables, and hack the system to progress.

Rogue AI

Face Nori, an AI entity that has gained sentience. She interacts with your desktop files and breaks the fourth wall.

Immersive Sim

Explore a detailed, atmospheric environment where every object tells a story. No hand-holding.
